# Data Tools Quick Reference

URL patterns for fetching documentation context.

## Airflow Providers

**URL pattern:**
```
https://airflow.apache.org/docs/apache-airflow-providers-{provider}/stable/_api/airflow/providers/{provider}/operators/{module}/
```

| Feature | Provider | Module |
|---------|----------|--------|
| HITL / Human-in-the-Loop | `standard` | `hitl` |
| Snowflake | `snowflake` | `snowflake` |
| Databricks | `databricks` | `databricks` |
| AWS S3 | `amazon` | `s3` |
| AWS Lambda | `amazon` | `lambda` |
| BigQuery | `google` | `bigquery` |
| GCS | `google` | `gcs` |
| Azure Blob | `microsoft-azure` | `wasb` |
